# DEVCONF 2026

A landing page for the DevConf 2026 developer conference, built with HTML5 and CSS3 only.

## Project Structure

- `index.html` — main page markup with the full conference layout.
- `style/styles.css` — global page styling and layout rules.
- `assets/` — image assets used in the navbar, hero, speaker cards, and footer.

## Page Sections

1. Navbar
2. Hero / Banner
3. Meet the Speakers
4. Secure Your Spot (Pricing)
5. AI Challenge / Launchpad
6. Footer

## Key Features

- Responsive layout using CSS grid and flexbox
- No JavaScript or external frameworks
- Full-width hero banner with overlay and CTA button
- Speaker grid with individual cards and track labels
- Pricing cards with a highlighted featured plan
- Custom AI challenge section designed for DevConf theme
- Footer with logo and social icon links
